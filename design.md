# Portfolio Website Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) and pure white (#ffffff) for high contrast
- **Accent**: Warm amber (#f59e0b) for highlights and interactive elements
- **Supporting**: Muted gray (#6b7280) for secondary text and subtle elements
- **Background**: Consistent dark theme with automatic light mode adaptation

### Typography
- **Display Font**: "Playfair Display" - Bold serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Code Font**: "JetBrains Mono" - For technical elements and code snippets
- **Hierarchy**: Large, impactful headings with smaller, refined body text

### Visual Language
- **Aesthetic**: Modern editorial style inspired by high-end tech publications
- **Mood**: Professional, confident, sophisticated
- **Approach**: Clean minimalism with strategic use of visual elements
- **Imagery**: High-quality construction and architecture photography

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations and micro-interactions
- **Splide.js**: Elegant project showcase carousel
- **ECharts.js**: Clean data visualization for skills
- **p5.js**: Subtle background particle effects
- **PIXI.js**: Advanced visual effects for hero section

### Effects & Animations
- **Text Effects**: 
  - Color cycling emphasis on key phrases
  - Split-by-letter stagger animations for headings
  - Subtle gradient text animation on hero title
- **Image Effects**: 
  - Ken Burns pan/zoom on hero images
  - Smooth hover transitions with 3D tilt effects
- **Background**: 
  - Subtle particle system using p5.js
  - Aurora gradient flow effect in hero section
- **Interactions**: 
  - Smooth scroll animations with proper fallbacks
  - Hover effects with depth and shadow
  - Form field focus animations

### Styling Approach
- **Layout**: CSS Grid and Flexbox for responsive design
- **Transitions**: Smooth, purposeful animations (150-300ms)
- **Shadows**: Subtle depth with layered shadows
- **Borders**: Minimal use, focus on spacing and typography
- **Responsive**: Mobile-first approach with progressive enhancement

### Header Effect
- **Background**: Liquid-metal displacement effect using PIXI.js
- **Content**: Large, bold typography with animated color cycling
- **Layout**: Centered content with proper spacing and visual hierarchy

### Theme Adaptation
- **Detection**: Automatic system preference detection
- **Transition**: Smooth theme switching without user interaction
- **Consistency**: Maintains design integrity across both themes
- **Accessibility**: High contrast ratios in both light and dark modes
